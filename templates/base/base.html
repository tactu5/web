Tabla de Frecuencia

¿Cómo funciona la tabla?

Introduce los datos de tu interés en el recuadro de 'DATOS', separa cada dato con una coma (,) o un espacio. Posteriormente escribe la cantidad de decimales que quieres que se manejen durante el calculo (se recomienda que sea la cantidad de digitos decimales que tienen la mayoria de los numeros). Una vez que introduzcas todos los datos presiona el botón de empezar y obtendras una Tabla de Distribucion de Frecuencias con base en los datos que introdujiste.

CLASES: Son subconjuntos de datos en los que se dividen los datos totales de la tabla.

Límite inferior [LI]: Es el valor menor de cada intervalo. A partir de la segunda clase se obtiene sumándole el la amplitud de clase al límite inferior anterior.

Límite superior [LS]: Es el número mayor de cada intervalo. Para obtener el primer dato de este apartado se debe aplicar “AMPLITUD DE CLASE - 1” a partir del segundo dato ya no aplica esto y se suma la cantidad completa.

Límite real inferior [LRI]: Es el valor menor de cada intervalo pero el límite real no presenta discontinuidad entre sus intervalos.

Límite real superior [LRS]: Es el valor mayor de cada intervalo pero el límite real no presenta discontinuidad entre sus intervalos.

Marca de clase [Xi]: Es el valor medio de cada clase; Se obtiene sumando el límite real superior y el límite real inferior y dividiéndolo entre dos.

Frecuencia absoluta [fi]: Es la cantidad de datos total por clase; Se obtiene contando la cantidad de datos que están siendo abordados por el intervalo (Límite inferior-límite superior).

Frecuencia relativa [hi]: Es la representación de manera porcentual de la cantidad individual de datos por clase; Se obtiene multiplicando la frecuencia absoluta de la clase por cien y dividiéndola entre el total de datos.

Frecuencia absoluta acumulada [Fi]: Es la sumatoria de todos los datos de la frecuencia absoluta; Se obtiene individualmente en cada clase, sumando la frecuencia absoluta anterior más la frecuencia absoluta de la clase.

Frecuencia relativa acumulada [Hi]: Es la sumatoria de todos los porcentajes de frecuencia relativa; Se obtiene individualmente en cada clase, sumando la frecuencia relativa anterior más la frecuencia absoluta de la clase.

Media aritmética [Xi * fi]: Conocida comúnmente como promedio se calcula sumando todas las mediciones y dividiéndolas entre el número total de datos por cada clase.

Desviación media [Xi - X]: Es la media aritmética de los valores absolutos de las desviaciones de los datos de las clases individuales respecto a la media aritmética. Entendido de forma sencilla, es el promedio de las desviaciones de las clases individuales.

D * f: Es el producto de la desviación media (Xi-X̅) y fi (frecuencia absoluta).

(Xi - X)2: Es la exageración de la desviación media, para lograr esto se eleva al cuadrado.

D2 * fi: Es el producto de la exageración de la desviación media y la frecuencia absoluta.

Amplitud de clase: Es la amplitud de cada clase; Se obtiene dividiendo el rango entre el número de clases.

Rango: Son todos los datos que se encuentran en un intervalo; Se obtiene restando al dato mayor el dato menor de cada conjunto de datos.

Desviación media: Es la media aritmética de los valores absolutos de las desviaciones medias de todos los datos respecto a la media aritmética.

Varianza: Es la media aritmética de los cuadrados de las desviaciones.

Media: Es el número más representativo de un conjunto de números (promedio).
